## 프로젝트 정리
- if (조건1) { if(조건2) { } else { } return; } else { } 에서, return이 없으면 조건1이 true더라도 조건1의 else문 실행됨. 이거 몰라서 콘솔창 중복이 심하게 났음.

- goto문 - 해당 메서드 내에서만 가능. 반복하고 싶은 위치에 자료형 없는 변수처럼 라벨명 + :(콜론), 이후 goto 라벨명으로 반복 조절.
